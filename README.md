## Run Locally

Clone the project

```bash
  git clone https://github.com/muhashfar/PersonalPortfolio.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  bundle install
```

Start the server

```bash
  bundle exec jekyll serve
```
